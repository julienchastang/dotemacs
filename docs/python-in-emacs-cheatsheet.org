#+TITLE: Python in Emacs Cheat Sheet

| Action                                           | Keybinding    | Command                              | Notes                                                                                                                                      |
| Open a Python file                               | =C-x C-f=     | =find-file=                          | Opens =.py= and activates Python mode; your hook on =python-base-mode= starts Eglot + Company + Ruff format-on-save.                       |
| Reconnect language server                        | =M-x=         | =eglot-reconnect=                    | Explicitly mentioned in your setup steps if LSP did not attach on first open.                                                              |
| Python env bootstrap (shell)                     | outside Emacs | =python3 -m venv .venv=              | First-time project setup in your notes.                                                                                                    |
| Activate env (shell)                             | outside Emacs | =source .venv/bin/activate=          | Needed before installing project/language tools.                                                                                           |
| Install project deps (shell)                     | outside Emacs | =pip install -r requirements.txt=    | Run inside the venv.                                                                                                                       |
| Install Python language tools (shell)            | outside Emacs | =pip install basedpyright ruff=      | Your server priority is basedpyright, then pyright, then pylsp.                                                                            |
| Optional LSP fallback (shell)                    | outside Emacs | =pip install python-lsp-server=      | Used only if pyright/basedpyright binaries are unavailable.                                                                                |
| Project env auto-load (shell)                    | outside Emacs | =direnv allow=                       | Your Emacs runs =direnv-mode=, so this wires project env vars into Emacs/LSP/REPL.                                                         |
| Format buffer                                    | =C-c C-f=     | =eglot-format-buffer=                | Custom keybinding in =eglot-mode-map=.                                                                                                     |
| Format on save                                   | automatic     | =ruff-format-on-save-mode=           | Enabled in your Python hook; runs on every save.                                                                                           |
| Organize imports                                 | =C-c C-o=     | =eglot-code-action-organize-imports= | Custom keybinding in =eglot-mode-map=.                                                                                                     |
| Rename symbol                                    | =C-c C-r=     | =eglot-rename=                       | Custom keybinding in =eglot-mode-map=.                                                                                                     |
| Jump to definition                               | =M-.=         | =xref-find-definitions=              | Works via Eglot. You configured xref display through Consult UI.                                                                           |
| Jump back from definition                        | =M-,=         | =xref-pop-marker-stack=              | Standard xref back navigation.                                                                                                             |
| Find references                                  | =M-?=         | =xref-find-references=               | Uses LSP backend when Eglot is active.                                                                                                     |
| Toggle symbol outline                            | =C-c i=       | =imenu-list-smart-toggle=            | Opens right-side imenu list for quick navigation in large Python files.                                                                    |
| Jump to function/class by name                   | =M-x=         | =imenu=                              | Minibuffer jump to definitions in the current Python buffer.                                                                               |
| Previous function definition                     | =C-M-a=       | =beginning-of-defun=                 | Move to the start of the current/previous def or class.                                                                                    |
| Next function definition                         | =C-M-e=       | =end-of-defun=                       | Move to the end of the current/next def or class.                                                                                          |
| Narrow by function name in buffer                | =M-s l=       | =consult-line=                       | Type part of a function name to jump quickly among definitions.                                                                            |
| Search in current buffer                         | =M-s l=       | =consult-line=                       | Fast in-buffer filtering.                                                                                                                  |
| Search project text                              | =M-s r=       | =consult-ripgrep=                    | Great for symbol/usage discovery across Python project files.                                                                              |
| Go to line                                       | =M-g g=       | =consult-goto-line=                  | Faster line jump UI.                                                                                                                       |
| Buffer switch                                    | =C-x b=       | =consult-buffer=                     | Better buffer/file switching.                                                                                                              |
| Jump to errors (next)                            | =M-g n=       | =next-error=                         | Works for Flymake diagnostics and other error sources.                                                                                     |
| Jump to errors (prev)                            | =M-g p=       | =previous-error=                     | Pairs with =next-error=.                                                                                                                   |
| Show diagnostics list                            | =M-x=         | =flymake-show-buffer-diagnostics=    | Your Python diagnostics are Flymake-based via Eglot (not Flycheck).                                                                        |
| Explain error at point                           | =M-x=         | =flymake-show-diagnostic-at-point=   | Shows full message for the current warning/error.                                                                                          |
| Apply quick fix/code action                      | =M-x=         | =eglot-code-actions=                 | Useful for import fixes and server-suggested edits.                                                                                        |
| Check diagnostics backend choice                 | n/a           | n/a                                  | This config uses Eglot/Flymake for Python. =flycheck= is listed as a package but not enabled/configured here.                              |
| Start Python REPL                                | =M-x=         | =run-python=                         | Not custom-bound, but should use project interpreter if direnv loaded env vars.                                                            |
| Send region to REPL                              | shadowed      | =python-shell-send-region=           | In many Python setups this is =C-c C-r=, but your =eglot-mode-map= binds =C-c C-r= to =eglot-rename=; run this via =M-x= or a custom bind. |
| Send buffer to REPL                              | =C-c C-c=     | =python-shell-send-buffer=           | Evaluate whole module quickly.                                                                                                             |
| Insert snippet                                   | =C-c y i=     | =yas-insert-snippet=                 | Your custom keybinding; uses minibuffer completion prompt.                                                                                 |
| Create new snippet                               | =C-c y n=     | =yas-new-snippet=                    | Your custom keybinding.                                                                                                                    |
| Visit snippet file                               | =C-c y v=     | =yas-visit-snippet-file=             | Your custom keybinding for editing snippets.                                                                                               |
| Expand snippet at point                          | =TAB=         | =yas-expand=                         | =yas-global-mode= is enabled and local snippet dir is =~/.emacs.d/snippets=.                                                               |
| Custom Python snippets found in this repo/config | n/a           | n/a                                  | None found under =/workspace=. No local snippet files/triggers were present here.                                                          |
